## 🚀 Deploy in 3 Steps

### Step 1: Create GitHub Repository

1. Go to: https://github.com/new
2. **Repository name:** `family-tree` (or your choice)
3. **Visibility:** Public ✓
4. **DON'T** check "Initialize with README"
5. Click **Create repository**

### Step 2: Run Deployment Script

```bash
cd /Users/Sachin.Deora/Documents/Personal/Family_Tree/code
./deploy-to-github.sh
```

Follow the prompts - it will:
- ✓ Initialize git
- ✓ Add your files
- ✓ Commit changes
- ✓ Push to GitHub

### Step 3: Enable GitHub Pages

1. Go to: `https://github.com/YOUR-USERNAME/family-tree/settings/pages`
2. Under "Build and deployment":
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
3. Click **Save**
4. Wait 1-2 minutes

**Your site will be live at:**
`https://YOUR-USERNAME.github.io/family-tree/`
